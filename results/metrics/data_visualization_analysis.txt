5. KEY FINDINGS
--------------------------------------------------------------------------------
• VITAL SIGNS: Extremely weak correlation with posture
  - Maximum correlation: 0.0059
  - Average correlation: 0.0027
  - Conclusion: Vital signs CANNOT predict posture effectively

• IMU SENSORS: MUCH stronger correlation with posture
  - Maximum correlation: 0.0766
  - Average correlation: 0.0335
  - Improvement: 12.9x stronger than vitals
  - Conclusion: IMU data is ESSENTIAL for posture prediction

• CLASS IMBALANCE: 9 classes underrepresented
  - May affect model performance on minority classes

6. RECOMMENDATIONS FOR MENTOR DISCUSSION
--------------------------------------------------------------------------------
PRIORITY 1 - INTEGRATE IMU DATA (CRITICAL):
  • IMU sensors show 12.9x stronger correlation
  • Current 11.26% accuracy is due to using only vital signs
  • Expected accuracy with IMU data: 60-85%
  • Top IMU features to use:
    1. Yaw_S1: 0.077 correlation
    2. Roll_S3: 0.056 correlation
    3. Pitch_S1: 0.049 correlation
    4. Roll_S1: 0.039 correlation
    5. Pitch_S2: 0.026 correlation

PRIORITY 2 - ADDRESS CLASS IMBALANCE:
  • Use class weights in model training
  • Consider SMOTE for minority class oversampling
  • Stratified sampling is already implemented

PRIORITY 3 - FEATURE ENGINEERING:
  • Create derived features from IMU sensors
  • Temporal features (movement patterns over time)
  • Angular velocity/acceleration from Roll/Pitch/Yaw

PRIORITY 4 - MODEL ARCHITECTURE:
  • Current MLP is adequate, but consider:
  • CNN for spatial patterns in sensor data
  • LSTM/RNN for temporal patterns
  • Ensemble methods

8. CONCLUSION
--------------------------------------------------------------------------------
The data visualization confirms that:

1. Low model accuracy (11.26%) is NOT due to model architecture problems
2. Low accuracy is due to INSUFFICIENT FEATURES (only vital signs used)
3. Vital signs have virtually NO predictive power for posture
4. IMU sensors are ESSENTIAL - they show 12.9x stronger correlation
5. Next step: MUST integrate IMU data to achieve acceptable accuracy

This is a valuable research finding: Human vital signs (temp, BP, SpO2) are
independent of posture, which makes biological sense. Posture is a mechanical
phenomenon best measured by IMU sensors (accelerometers, gyroscopes).
